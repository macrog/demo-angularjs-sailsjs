This is a demo fro a Single Page Application using Sails.js (back-end) and AngularJs(front-end).

The goal is to demonstrate a simple CRUD operation on employee record.

Sails.js is connected to mongoLab database.

On success connection there sould be one record display in employee table.

Front-end technologies use: Boostrap3, Ui-Boostrap and Angular Material.

Layout is fully responsive.
